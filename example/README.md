# Example

Adds a darkmode button to the theme `stockholm`.\
Uses [Darkmode.Js](https://darkmodejs.learn.uno/) and [zepto.js](https://zeptojs.com/).

## Prerequisites

Knowledge about the [Yellow Folder Structure](https://datenstrom.se/yellow/help/api-for-developers#folder-structure).

## Installation

  - Install this extension.
  - Copy all files / directories from the example to a fresh Yellow installation.
